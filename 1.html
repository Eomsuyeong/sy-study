HTML

구조화

!doctype html : 이 웹페이지가 HTML로서 만들어졌다는 코드
html : body태그와 head태그를 감싸는 태그
title : 브라우저의 제목 표시줄이나 페이지 탭에 보이는 문서 제목
head : 본문을 설명하는 태그
body : 본문
저장 단축기 : ctrl + S

태그

strong - 글씨를 굵게
u - 밑줄
h1~h10 - 제목
br - 줄바꿈
p - 단락
img - 이미지 삽입 / src="파일이름" width="크기"
단락 간격조절 - p style="margin-top:크기px;"
a - 링크걸기
a는 링크를 거는 것이고 href(HyperText Reference)는 '웹브라우저야, 이 링크는 이 값을 참조해'라는 뜻
target="_blank" - 링크를 클릭했을 때 새창에서 페이지가 열리게 하는 속성
title은 이 링크가 어떤 내용을 담고 있는지를 툴팁으로 보여주는 기능
